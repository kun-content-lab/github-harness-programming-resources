# Public Boundary

This repo is designed to be copied by other people. Keep it reusable and safe.

## Allowed

- Workflow descriptions.
- Public Skill examples.
- Project instruction examples.
- GitHub templates.
- Checklists.
- Demo projects with invented data.
- General diagrams.

## Not Allowed

- Credentials, cookies, or session material.
- Personal workspace paths.
- Private project materials.
- Account-specific automation.
- Internal-only scripts that cannot run outside the original environment.
- Raw issue, PR, or comment text from non-public projects.

## Rewrite Rule

If something is useful but environment-specific, rewrite it into a public pattern.

| Original kind | Public version |
|---|---|
| Local startup prompt | `AGENTS.example.md` with placeholders |
| Private Skill | Public Skill that describes the workflow |
| Private issue template | Generic GitHub template |
| Project-specific script | Checklist or pseudocode |
| Real project record | Demo example with invented data |
