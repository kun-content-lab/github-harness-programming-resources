# Public Boundary Checklist

Before publishing or copying this kit, check:

- [ ] No credentials, cookies, or session material.
- [ ] No personal workspace paths.
- [ ] No private project material.
- [ ] No account-specific automation.
- [ ] No raw issue or PR text from non-public projects.
- [ ] Environment-specific logic is rewritten as a public pattern.
- [ ] Examples use invented or public demo data.
