# Workflow: Review And Close Loop

Use this when an agent claims work is ready.

## Steps

1. Read the evidence comment.
2. Check direction against the original demand.
3. Check boundary against the task issue.
4. Open the evidence links or files.
5. Decide next step:
   - close;
   - continue;
   - split;
   - return to Discussion.
6. Record the decision in a comment.

## Output

- A clear close or continue decision.
- Follow-up issues when needed.

## Stop Condition

If evidence cannot be reviewed, keep the issue open.
